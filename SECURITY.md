# Security Policy

## Reporting a Vulnerability
If you discover a security vulnerability in Soulkey,
please email **security@soulkey.world** immediately.

Do not disclose publicly until the issue has been fixed.

## Supported Versions
- `main` branch only

## Security Practices
- 2FA enforced for all maintainers
- Secret scanning and Dependabot enabled
- npm publish uses 2FA and provenance
